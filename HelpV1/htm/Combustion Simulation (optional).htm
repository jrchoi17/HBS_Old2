Fig. 2.1-1: 열풍로 가스 계산 인터페이스

2.1.1 키워드

  1. Blast Furnace Gas (BFG): 용광로의 가스 부산물
  2. Coke Oven Gas (COG): 석탄을 코크스로 전환할 때 나오는 가스
  3. X-gas: 연소 과정에 추가된 임의의 가스 조성(LPG 가스, 등)
  4. M-gas: BFG/COG/X-Gas의 Mixing mole ratio에 따라 도출되는 Species별 mole/mass fraction

2.1.2 사용자 입력

해석에 필요한 구성요소의 정보를 을 눌러 불러올 수 있다. 만약 정보를 불러오지 않을 시 프로그램 내부에 저장되어 있는 default 값으로 계산이 진행된다. 불러온 정보를 사용하고 싶지 않을 시 을 누르면 default 값으로 초기화 된다. 반대로 저장된 값을 밖으로 불러오기 위해서는 버튼을 눌러 저장이 가능하다.

2.1.3 가스 조건 설정

테이블 왼쪽 상단에 위치한 들을 클릭하여 계산에 사용될 가스 종류를 선택할 수 있으며 중복선택이 가능하다. 선택된 가스 종류가 섞여있는 비율은 테이블 아래에 위치한 칸에 가스별로 각각 입력이 가능하다. 입력값은 예시와 같이 총 합이 100이 되야한다.만약 BFG/COG/X-Gas 값이 100을 초과하거나 미만일 때 을 눌러 값들을 100으로 표준화 시킬 수 있다.

2.1.4 사용자 출력

출력될 결과는 중 하나를 선택하여 M_Gas열에 Mole Fraction 아니면 Mass Fraction이 출력될지 선택이 가능하다. 를 누르면 설정조건에 따른 열풍로 내부 유체의 mole/mass fraction이 계산된다. 계산된 결과 값은 테이블 우측에 그래프 형태로 시각화 된다.
